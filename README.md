# 🎬 Kinopoisk Fullstack

Fullstack-приложение для просмотра топ-250 фильмов с открытого API Кинопоиска.  
Проект состоит из Vue-фронтенда и Express-прокси для работы с API.

## 🛠️ Технологии

**Frontend:**  
- Vue 3 (Composition API)  
- Axios  
- Bootstrap (адаптивная вёрстка)  

**Backend:**  
- Node.js  
- Express  
- Axios (прокси-запросы)  

**API:** Kinopoisk API Unofficial  

## 🚀 Функционал

- Получение топ-250 фильмов с рейтингом и постерами  
- Отображение карточек фильмов (название, год, рейтинг)  
- Прокси-слой на бекенде (скрывает API-ключ)  
- Адаптивная сетка карточек
